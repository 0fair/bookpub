# Book Download Site

Простой статический сайт на GitHub Pages для автоматического скачивания книги.

## Архитектура

```
/
├── index.html      # Главная страница с автоскачиванием
├── publish/        # Папка с книгами (первая книга скачивается автоматически)
│   └── book.epub   # (или .pdf, .fb2 — любой формат)
└── CLAUDE.md
```

## Требования

- При открытии сайта автоматически начинается скачивание первого файла из `./publish`
- Минималистичный дизайн: сообщение о скачивании + ручная ссылка на случай если автоскачивание не сработало
- Никаких зависимостей, фреймворков, сборщиков — чистый HTML/CSS/JS (по возможности)
- Makefile с командой publish

## Команды

```bash
make publish
```

команда должна запушить в репозиторий все файлы и опубликовать новый сайт на github pages через утилиты git и gh

## Частые проблемы

| Проблема                 | Решение                                                                                 |
|--------------------------|-----------------------------------------------------------------------------------------|
| 404 на книгу             | Проверь путь в на книгу и если ее нет - выдай ошибку пользователю, что книга не найдена |
| Скачивание не начинается | Некоторые браузеры блокируют автоскачивание — поэтому есть ручная кнопка                |
| Pages не включается      | Репозиторий должен быть public (или GitHub Pro для private)                             |
